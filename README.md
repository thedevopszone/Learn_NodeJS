# Learn NodeJS


```
mkdir Kindle
cd Kindle

npx create-next-app@latest next-hello-world
cd next-hello-world

# For Prod
npm run build

# For Dev
npm run dev

npm run start

```


## Install Tailwind

```
npm install -D tailwindcss postcss autoprefixer
```

Create tailwind config files (postcss.config.js and tailwind.config.js)
```
npx tailwindcss init -p
```
